📌 **Project Overview**

This project focuses on performing end-to-end sales analysis using SQL and Power BI. The objective is to extract meaningful business insights from the Global Superstore dataset and build an interactive dashboard for decision-making.

**🛠️ Tools & Technologies**

SQL (Oracle SQL Developer)

Power BI

Excel

**🔍 Data Preparation**
Cleaned column names (removed special characters)

Fixed data type issues (dates, numeric fields)

Handled missing/invalid columns

Ensured structured and consistent dataset

**📈 Exploratory Data Analysis (EDA)**

Performed analysis using SQL to understand data patterns:

Total Sales, Profit, and Order Count

Sales by Category and Region

Top 5 Products by Sales

Loss-making Products

Customer-level analysis

**📊 Dashboard (Power BI)**

🔹 Page 1: Sales Overview

KPI Cards: Total Sales, Total Profit, Total Quantity, Profit Margin

Monthly Sales Trend

Sales by Category

Filters: Region, Date, Category

🔹 Page 2: Detailed Analysis

Regional Sales Performance

Segment-wise Sales Distribution

Shipping Mode Analysis

Detailed Table (Customer, Product, Sales, Profit, Quantity)

**🎯 Key Insights**

Identified top-performing categories and regions

Detected loss-making products impacting profitability

Observed sales trends over time

Highlighted customer and segment contributions

**💡 Business Value**

This dashboard helps stakeholders:

Monitor overall business performance

Identify profitable and underperforming areas

Make data-driven decisions for growth and optimization

**🚀 Conclusion**

This project demonstrates strong skills in data cleaning, SQL-based analysis, and Power BI dashboard development to convert raw data into actionable insights.
